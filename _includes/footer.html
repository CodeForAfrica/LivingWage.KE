Living Wage Kenya calculator lets employers calculate if they are paying their employees a living wage.

The tool was created by OpenUp, formerly Code for South Africa, to highlight the low wages earned by domestic workers.

This site is an africanSPENDING project of Code for Africa. All content is released under a Creative Commons 4 Attribution license. Reuse it to help empower your own community. The code is available on GitHub and data is available on ​openAFRICA​.
